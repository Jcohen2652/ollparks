# Frontend OLL PARKS

## Stack

- Next.js 15 (App Router) + TypeScript
- React Server Components
- TailwindCSS + shadcn/ui
- Mapbox GL JS (cartographie)
- TanStack Query (data fetching)
- Zustand (state local)

## Pages prévues

- `/` — dashboard (KPIs, opportunités du jour, RDV)
- `/entreprises` — fiches entreprises
- `/contacts` — fiches contacts
- `/biens` — catalogue offres + carte
- `/besoins` — demandes en cours
- `/matching` — top opportunités scorées
- `/visites` — agenda visites + génération docs
- `/documents` — moteur de docs dynamiques
- `/outlook` — explorateur emails enrichi
- `/proprietaires` — base propriétaires (Pappers Immo)
